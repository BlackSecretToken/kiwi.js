Kiwi.Buffers.Bufferer Class

Defined in: src/buffers/bufferer.ts:11
Module: Buffers
Parent Module: Kiwi

Available since 1.5.0

Master system for rendering frame buffers.

You should use a Bufferer to create and control all buffers within a State, rather than create them singly. This ensures that GroupBuffer and FilterBuffer objects are properly registered.
